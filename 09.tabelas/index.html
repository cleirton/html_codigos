Usando tabelas

Tabela simples

A tabela tem 3 principais tags:

  • table = tag principal para definir a tabela
  • tr = table row, dentro desta tag estão os elementos da linha
  • td = table cell, aqui vai o valor propriamente dito

Tabela abaixo sem nenhuma formatação:

Peça Cor Qtde
Short jeans Azul 120
Conjunto Maria Verde 150
Vestido Ju Branco 128
Calça Jeans Azul escuro 148

Tabela com Headers

Podemos usar na primeira linha th ao invés de td (há uma mudança na formatação da primeira linha):

Peça Cor Qtde
Short jeans Azul 120
Conjunto Maria Verde 150
Vestido Ju Branco 128
Calça Jeans Azul escuro 148
