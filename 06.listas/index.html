Criando algumas listas

Ordered List, lista ordenada

Uma lista ordenada simples

  1. Primeiro item
  2. Segundo item
  3. Terceiro item
  4. Quarto item
  5. Quinto item

Mudando o parâmetro type para modificar o marcador (type="1")

  1. Primeiro item
  2. Segundo item
  3. Terceiro item
  4. Quarto item
  5. Quinto item

(type="A")

  5. Quinto item

(type="a")

  1. Primeiro item
  2. Segundo item

  4. Quarto item
  5. Quinto item

(type="I")

  1. Primeiro item
  2. Segundo item

  4. Quarto item
  5. Quinto item

(type="i")

  1. Primeiro item

  5. Quinto item

Iniciando a lista por um valor diferente de 1(type="1" start="3")

Unordered lists, listas não ordenadas

Uma lista não ordenada simples

  • Item 1
  • Item 2
  • Item 3
  • Item 4

Mudando a propriedade type podemos mudar o marcador (type="disc")

  • Item 2
  • Item 3
  • Item 4

(type="circle")

(type="square")

  • Item 1

  • Item 3
  • Item 4

Listas aninhadas

Podemos criar listas dentro de outras listas, listas internas recebem um recuo automático

  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
    3. Item 2.3
  3. Item 3
  4. Item 4
