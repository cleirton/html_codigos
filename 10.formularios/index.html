Principais tags:

  • label, como se fosse o "nome" do campo.
  • input, onde o usuário modifica o campo.

Para ligar a tag label a tag input é preciso que o atributo for da tag label seja igual ao atributo id da tag input




OBS:O atributo id é único para cada tag, mais na frente veremos que o atributo class tem uma função parecida com o atributo id
